## Expense Tracker

A simple expense tracker application built using MERN Stack.

### Installation

```bash
git clone git@github.com:MehulBawadia/mern-expense-tracker.git

# For Backend
cd mern-expense-tracker/backend
cp .env.example .env
npm install
npm run dev

# For Frontend
cd mern-expense-tracker/frontend
npm install
npm run dev
```
